Larisdy.com
Larisdy adalah website e-commerce sederhana untuk produk sambal premium Indonesia. Project ini dibuat sebagai tugas kolaborasi menggunakan Git dan GitHub dengan sistem branching dan pull request.

Fitur Utama
- Sistem Login menggunakan localStorage
- Keranjang Belanja
- Checkout dan Simulasi Pemesanan
- Halaman Akun Pengguna
- Kategori Produk
- Indikator Tingkat Kepedasan Produk
- Penyimpanan data user, cart, dan order di localStorage

Teknologi yang Digunakan
- HTML
- CSS
- JavaScript 
- LocalStorage
- VSCODE

 Struktur Folder

Larisdy.com/

│

├── docs/ → Dokumentasi proyek

├── src/ → File utama (HTML, CSS, JS)

├── tests/ → Pengujian

└── README.md

Pembagian Tugas

- Pengembangan Struktur HTML (Marvelio)
- Pengembangan Styling (Tatan)
- Pengembangan Fitur Interaktif (Selly)
- Dokumentasi (Docs / Technical Writer) ( Isha)

Alur Kolaborasi

1.  Clone repository
2.  Membuat branch fitur
3.  Commit perubahan
4.  Push ke branch masing-masing
5.  Membuat Pull Request
6.  Review \& Merge ke branch main



Project ini bersifat simulasi dan tidak menggunakan backend/server. Karena semua data disimpan di browser menggunakan localStorage.
