\# Larisdy.com



Larisdy adalah website e-commerce sederhana untuk produk sambal premium Indonesia. 

Project ini dibuat sebagai tugas kolaborasi menggunakan Git dan GitHub dengan sistem branching dan pull request.



---



\## 🚀 Fitur Utama



\- 🔐 Sistem Login \& Register (menggunakan localStorage)

\- 🛒 Keranjang Belanja Dinamis

\- 📦 Checkout dan Simulasi Pemesanan

\- 👤 Halaman Akun Pengguna

\- 🔍 Filter Kategori Produk

\- 🌶️ Indikator Tingkat Kepedasan Produk

\- 💾 Penyimpanan data user, cart, dan order di localStorage



---



\## 🛠️ Teknologi yang Digunakan



\- HTML5

\- CSS3 (Custom Styling \& Animations)

\- JavaScript (Vanilla JS)

\- LocalStorage (Simulasi Database)



---

\## 🔀 Alur Kolaborasi



1\. Clone repository

2\. Membuat branch fitur

3\. Commit perubahan

4\. Push ke branch masing-masing

5\. Membuat Pull Request

6\. Review \& Merge ke branch main



---



\## 📌 Catatan



Project ini bersifat simulasi dan tidak menggunakan backend/server.

Semua data disimpan di browser menggunakan localStorage.
